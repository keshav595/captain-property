# Captain Property 🏡

A modern real estate landing page for property listings and lead generation.

## ✨ Features

- Clean and modern UI
- Responsive layout for all screen sizes
- Property showcase section
- Contact form for lead generation
- Fast-loading static website

## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content, images, and contact details as needed

- Lead generation website for property dealers
